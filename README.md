# US National Debt Analysis & Forecasting
US national debt analysis using Treasury data with trend identification, debt segmentation, and forecasting of long-term growth

## Overview

This project analyses US national debt trends using historical Treasury data. The dataset was cleaned, structured, and analysed to identify growth patterns and build future projections.

## Objectives

* Analyse large-scale financial data
* Identify trends and anomalies
* Forecast future debt growth

## Dataset

* Source: US Treasury data
* Time period: Dec 2022 – Feb 2023
* Total debt range: **$31.3Tn – $31.5Tn**

## Key Highlights

* Segmented debt into:

  * **Public Debt (~$24.5Tn)**
  * **Intragovernmental Holdings (~$6.8Tn)**
* Identified sharp slowdown in growth:

  * **6.1% (2022) → 0.1% (early 2023)**
* Detected **~5.2% seasonal patterns**
* Forecasted public debt growth:

  * **$24.6Tn (2023) → $32.3Tn (2028)** (+31%)

## Analysis Performed

* Data cleaning and restructuring
* Time-series trend analysis
* Year-on-year variance analysis
* Pivot-based pattern detection
* Forecast modelling

## Tools Used

* Excel
* Python (Pandas, NumPy)
* Data analysis techniques

## Files Included

* Cleaned dataset
* Analysis workbook
* Forecast model
* Charts and visual outputs

## Key Insight

The slowdown in debt growth highlights how macroeconomic conditions can significantly impact fiscal trends, reinforcing the importance of data-driven monitoring in public finance.
